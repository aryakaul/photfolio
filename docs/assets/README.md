# Assets Directory

Place static assets for your portfolio site here.

## Favicon

Add your website icon (favicon) to this directory:
- Recommended: `favicon.ico` (classic format)
- Modern: `favicon.png` or `favicon.svg`
- Sizes: 32x32, 64x64, or SVG for scalability

Configure the path in `config.yaml`:
```yaml
site:
  favicon: "assets/favicon.png"
```

## Other Assets

You can also store:
- Logo images
- Custom fonts
- Other static files

These files will be copied to the build directory during the build process.
